# Network Topology Project

## Overview

The project focuses on designing and implementing a network topology which is simulated as a real world business environment. The aim was to have a reliable and efficient network that aids communication among various devices and adheres to the fundamental networking principles. With the assistance of such applications as Cisco Packet Tracer, which comprised routers, switches, and end devices, I have managed to develop a system network and offer sufficient connectivity and structure.

Throughout the project, I applied some of the best concepts of networking that involve IP addressing, subnetting and OSI model to create a fully functional and scalable system. I established devices to allow communication over the network and checked connectivity to make sure that everything was running properly. With this project, I was able to reinforce my knowledge of network design, data movement within networks, as well as troubleshooting of typical networking problems.

Overall, this project demonstrates that I am capable of designing, configuring, and analyzing network systems and applying the theoretical knowledge and practical technical skills.
## Key Features

* IP addressing and subnetting
* Router and switch configuration
* End-to-end connectivity testing
* Network segmentation and design

## Tools Used

* Cisco Packet Tracer
* TCP/IP Networking
* OSI Model concepts
